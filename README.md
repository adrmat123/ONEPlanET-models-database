# ONEPlanET WEF Nexus Models

System Dynamics models for Water-Energy-Food Nexus planning in African river basins. This repository contains the complete results from the **ONEPlanET** project (HORIZON-CL5-2021-D3-03-01), an EU-AU partnership developing open-source modelling tools for sustainable energy transition in Africa.

## Project Overview

ONEPlanET develops system dynamic models to support African policy-makers and stakeholders in understanding the interlinkages between water, energy, and food sectors. The models integrate climate change impacts, renewable energy pathways, and policy scenarios at river basin scale.

**Three case studies:**
- Songwe River Basin (Malawi/Tanzania)
- Inkomati-Usuthu Water Management Area (South Africa)
- Bani River Basin (Mali)

## Repository Structure

### Dataset

**Directory:** `Dataset/`

- `ONEPlanET_database.xlsx` – Integrated WEF Nexus dataset (supervised by CARTIF in WP2)
- `ONEplanET_dataset_documentation.pdf` – Technical documentation for the dataset

### Models

Each case study has its own directory with three files:

**Songwe:**
- `Songwe/Songwe_Model.mdl` – Vensim model
- `Songwe/Songwe_Model_Parameters.xlsx` – Model parameters
- `Songwe/Songwe_Scenario_Parameters.xlsx` – Scenario parameters

**Inkomati-Usuthu:**
- `Inkomati-Usuthu/Inkomati-Usuthu_Model.mdl` – Vensim model
- `Inkomati-Usuthu/Inkomati-Usuthu_Model_Parameters.xlsx` – Model parameters
- `Inkomati-Usuthu/Inkomati-Usuthu_Scenario_Parameters.xlsx` – Scenario parameters

**Bani:**
- `Bani/Bani_Model.mdl` – Vensim model
- `Bani/Bani_Model_Parameters.xlsx` – Model parameters
- `Bani/Bani_Scenario_Parameters.xlsx` – Scenario parameters

## Requirements

- Vensim (for opening and running .mdl files)
- Microsoft Excel or compatible spreadsheet software (LibreOffice Calc, etc.)
